# PackTrack

**Never Wait to Work Out**

PackTrack is a mobile app for Emory students to track busy times at campus gyms. This repository contains the project website documenting our user-centered design process for CS 485/584 Human-Computer Interaction.

## Project Website

The website showcases our design journey from initial pitch through final prototype, including:

- Problem identification and pitch (P1)
- User research findings (P2)
- Ideation and lo-fi prototypes (P3)
- Mid-fi prototype (P4)
- Heuristic evaluation (A4)
- Hi-fi prototype (P5)
- Final demo and report (P6)

## Tech Stack

- React 18 + Vite
- Tailwind CSS
- Lucide React icons
- GitHub Pages deployment

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Deployment

The site automatically deploys to GitHub Pages on push to main via GitHub Actions.
